关于漂流瓶

漂流瓶 v3.0

网页版 - AI增强版

版本信息
  • 当前版本: v3.0 网页版 AI增强版
  • 原始版本: v1.1 命令行版
  • 中间版本: v2.0 图形界面版
核心功能
  • 用户注册和登录（密码加密存储）
  • 发送和接收漂流瓶消息
  • 私聊功能（文本、图片、文件、语音消息）
  • 群组聊天功能（创建、管理、邀请链接）
  • 消息已读未读状态显示
  • 消息撤回功能
  • 好友搜索和添加
  • 群组创建和管理
  • 朋友圈功能（发布动态、点赞、评论）
AI智能功能 ⭐ NEW
  • AI自动回复: 忙碌时自动回复消息
  • AI好友: 与AI助手进行智能对话
  • 消息总结: AI智能总结未读消息
  • 回复生成: AI帮助生成聊天回复
  • 群组AI助手: 在群组中使用 /AI 命令
Mini Apps平台 ⭐ NEW
  • AI生成应用: 通过自然语言生成HTML5应用
  • 上传自定义应用: 上传自己开发的HTML应用
  • 应用管理: 查看、删除已创建的应用
  • 应用分享: 与好友分享有趣的Mini应用
实时通讯
  • WebSocket实时连接
  • 实时消息推送
  • 在线状态同步
  • 输入状态提示
安全特性
  • CSRF令牌保护
  • 密码哈希加密（pbkdf2:sha256）
  • 文件上传类型和大小限制
  • SQL注入防护
  • XSS攻击防护
技术栈
  • 后端: Python 3.13, Flask 2.3.2, SQLite, Flask-SocketIO 5.3.0
  • 前端: HTML5, CSS3, JavaScript ES6+, Bootstrap 5.3.0, jQuery 3.6.0
  • 实时通信: Socket.IO 4.0.1
  • AI集成: 第三方AI API
  • 数据库: SQLite
版本历史
  • v1.1 - 基础命令行版本，实现基本的漂流瓶功能
  • v2.0 - 图形界面版本，新增私聊和图片功能
  • v2.0 网页版 - 基于Flask的网页版本，新增群组功能
  • v3.0 - AI增强版，新增AI功能、Mini Apps平台、实时通讯等
v3.0 新特性
